About pure-flon Hypercasual Factory

Stack Tower is the first release from the pure-flon hypercasual game factory — a production pipeline that generates 1-2 prototypes per week, validates them through closed testing against KPI thresholds, and promotes survivors to public release.

All game development, asset generation, and build automation is driven by the HC-series nightly pipeline at 23:00~03:30 KST, backed by AI design (Visual/Sound), Unity headless build, and Google Play Publisher API integration.
