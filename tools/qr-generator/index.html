Frequently Asked Questions자주 묻는 질문

How do I create a WiFi QR code? WiFi QR 코드는 어떻게 만드나요? +

Select the WiFi tab, enter your SSID, password, and security type. Click Generate. Scanning the code on a phone will automatically connect to the network. WiFi 탭을 선택하고 네트워크 이름(SSID), 비밀번호, 보안 방식을 입력하세요. 생성 버튼을 누르면 완성됩니다. 스마트폰으로 스캔하면 자동 연결됩니다.

What is the difference between PNG and SVG? PNG와 SVG의 차이는 무엇인가요? +

PNG is a raster image at a fixed pixel size. SVG is scalable vector — it looks crisp at any size. Use SVG for print materials and PNG for digital sharing. PNG는 고정된 픽셀 크기의 래스터 이미지입니다. SVG는 벡터 형식으로 어떤 크기에서도 선명하게 표시됩니다. 인쇄물에는 SVG, 디지털 공유에는 PNG를 사용하세요.

Is this QR generator private? 개인 정보가 안전한가요? +

Yes. All processing is done in your browser. No data is sent to any server. Your WiFi password and personal details never leave your device. 예. 모든 처리는 브라우저에서 직접 이루어집니다. 서버로 전송되는 데이터가 없습니다. WiFi 비밀번호나 개인 정보는 기기 밖으로 나가지 않습니다.
